Jawaban No 1

Website adalah kumpulan halaman web yang dapat diakses melalui internet dengan menggunakan browser. Cara kerja website adalah sebagai berikut: Pengguna mengetikkan nama domain website di browser, misalnya www.bing.com. Browser mengirimkan permintaan ke DNS server untuk mencari alamat IP dari nama domain tersebut. DNS server mengembalikan alamat IP yang sesuai dengan nama domain, misalnya 204.79.197.200. Browser mengirimkan permintaan ke web server yang memiliki alamat IP tersebut untuk meminta file-file yang dibutuhkan untuk menampilkan halaman web, misalnya file HTML, CSS, JavaScript, gambar, dan lain-lain. Web server menerima permintaan dari browser dan mengirimkan file-file yang diminta melalui protokol HTTP atau HTTPS. Browser menerima file-file dari web server dan menampilkan halaman web di layar pengguna. Web server, domain, dan hosting adalah tiga komponen penting yang berhubungan dalam cara kerja website. Berikut penjelasannya: Web server adalah perangkat keras dan perangkat lunak yang bertugas untuk menyimpan, memproses, dan mengirimkan file-file website ke browser. Contoh web server adalah Apache, Nginx, IIS, dan lain-lain. Domain adalah nama unik yang digunakan untuk mengidentifikasi alamat website di internet, misalnya www.bing.com. Domain memudahkan pengguna untuk mengingat dan mengakses website tanpa harus mengetahui alamat IP-nya. Hosting adalah layanan penyewaan ruang penyimpanan di web server untuk menempatkan file-file website. Hosting memungkinkan website dapat diakses secara online oleh pengguna di seluruh dunia. Contoh penyedia hosting adalah Jagoan Hosting1, Dewaweb2, Nusa3, dan lain-lain.
